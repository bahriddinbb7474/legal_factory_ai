# MVP Scope

The first working version should focus on practical factory legal scenarios.

## First Working Scenarios

- Review local contracts.
- Review import contracts.
- Analyze client debts.
- Draft debt letters and claims.
- Analyze letters from the tax authority and other government bodies.
- Answer HR and labor questions.
- Generate letters, claims, memos, and preliminary legal opinions.

## Included in Early Scope

- Web chat.
- Three agent configuration.
- File upload.
- Right document editor.
- Source-required legal response format.
- Risk levels and approval flag.
- Basic roles.
- Usage and cost tracking.

## Postponed

- Complex court practice analytics.
- Full automatic daily law monitoring.
- ERP or 1C integration.
- Advanced workflow engine.
- Separate mobile application.
- Full legal expertise under foreign law.
